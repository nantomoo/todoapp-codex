# 機能一覧

## 1. 認証機能

| 機能ID | 機能名 | 概要 | 利用者 |
|---|---|---|---|
| AUTH-01 | 会員登録 | メールアドレスとパスワードで新規登録を行う | ゲスト |
| AUTH-02 | ログイン | 登録済みアカウントでログインする | ゲスト |
| AUTH-03 | ログアウト | 現在のセッションを終了する | 会員 |
| AUTH-04 | パスワード再設定 | パスワード再設定メールを送信する | ゲスト |
| AUTH-05 | セッション維持 | ログイン状態を維持し、認証必須画面を保護する | 会員 |

## 2. プロフィール機能

| 機能ID | 機能名 | 概要 | 利用者 |
|---|---|---|---|
| PROF-01 | プロフィール参照 | 自分のプロフィール情報を表示する | 会員 |
| PROF-02 | プロフィール更新 | 表示名、自己紹介を更新する | 会員 |

## 3. 投稿機能

| 機能ID | 機能名 | 概要 | 利用者 |
|---|---|---|---|
| POST-01 | 投稿一覧表示 | 公開済み投稿の一覧を表示する | ゲスト、会員 |
| POST-02 | 投稿詳細表示 | 投稿の詳細情報を表示する | ゲスト、会員 |
| POST-03 | 投稿作成 | 新規投稿を作成する | 会員 |
| POST-04 | 投稿更新 | 自分の投稿を編集する | 会員 |
| POST-05 | 投稿削除 | 自分の投稿を削除する | 会員 |
| POST-06 | 公開状態変更 | 投稿を下書き、公開、アーカイブに切り替える | 会員 |
| POST-07 | 自分の投稿一覧表示 | ログインユーザー自身の投稿一覧を表示する | 会員 |

## 4. 外部連携、サーバー処理

| 機能ID | 機能名 | 概要 | 利用者 |
|---|---|---|---|
| FUNC-01 | 外部 API 連携 | 秘密鍵を使う外部 API 呼び出しを行う | システム |
| FUNC-02 | Webhook 受信 | 外部サービスからの通知を受信する | システム |
| FUNC-03 | 通知送信 | 投稿公開時などの通知を送信する | システム |
| FUNC-04 | 監査ログ記録 | 重要操作を監査ログへ記録する | システム |
